# b4b-apps

Windows 向けの小さな Python ツールをまとめたリポジトリです。

## 含まれるアプリ

- `comment-sender`: 定型コメントを素早く送信する GUI ツール
- `player-jiggle`: `W / A / S / D` をランダムに入力する簡易ツール

## 各アプリの使い方

詳細はそれぞれの README を参照してください。

- [player-jiggle/README.md](player-jiggle/README.md)
