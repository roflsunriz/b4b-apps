# comment_sender

`comment_sender.pyw` は、ゲームやアプリに向けて定型コメントを素早く送信するための Windows 向け GUI ツールです。

## 特徴

- コメント一覧を GUI で管理
- 選択したコメントをクリップボード経由で送信
- 送信間隔と繰り返し回数を設定可能
- ホットキーで前後のコメントを切り替え可能
- 設定ファイルは自動生成

## 対応環境

- Windows
- Python 3.8 以上推奨

## インストール

1. このリポジトリを展開します。
2. 依存関係をインストールします。

```bash
pip install -r requirements.txt
```

`comment_sender.pyw` は起動時に必要なライブラリが不足している場合、自動でインストールを試みます。

## 起動方法

`comment_sender.pyw` をダブルクリックして起動します。

または、コマンドラインから次のように実行できます。

```bash
python comment_sender.pyw
```

## 使い方

1. 左側のコメント一覧から送信したいコメントを 1 件選択します。
2. 必要に応じて `送信間隔（秒）` と `繰り返し回数` を変更します。
3. `送信開始` を押すと送信します。
4. 送信を止めたい場合は `中止` を押します。

### コメントの編集

1. 一覧からコメントを選択します。
2. `編集` を押します。
3. テキストを修正して `保存` を押します。

### 設定の保存

`保存` を押すと、コメント一覧、送信間隔、繰り返し回数が設定ファイルに保存されます。

## ホットキー

- `j+8` : 選択中のコメントを送信
- `j+9` : 前のコメントを選択
- `j+0` : 次のコメントを選択
- `j+p` : `UI操作を無効化` の切り替え

## 設定ファイル

初回起動時、または設定ファイルがない場合に次のファイルが自動生成されます。

```text
misc/config/comment_sender_back4blood.json
```

このファイルには次の内容が保存されます。

- `comments`
- `interval`
- `repeat`

## UI操作を無効化

`UI操作を無効化` を有効にすると、ボタンや入力欄などの UI 操作を止められます。
ただし、ホットキー自体は引き続き利用できます。

## 注意

- 送信処理は `pyautogui` とクリップボード操作に依存します。
- 一部の環境では、ホットキー監視や入力制御のために管理者権限が必要になる場合があります。
- コメント送信先のアプリやゲームの利用規約に反しない範囲で使ってください。

## 依存関係

- `pyautogui`
- `keyboard`
- `clipman`
