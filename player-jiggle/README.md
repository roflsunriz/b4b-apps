# Player Jiggle

`Player Jiggle` は、指定した間隔ごとに `W / A / S / D` をランダムに押下する簡易ツールです。  
`Ctrl` を押しっぱなしにした状態で動作し、ゲームやアプリの入力補助を想定しています。

## 動作環境

- Windows
- Python 3.x

## 必要なもの

- `pyautogui`
- `keyboard`

`tkinter` は Python に標準で含まれているため、通常は追加インストール不要です。

## インストール

```bash
pip install -r requirements.txt
```

## 起動

```bash
python player_jiggle.pyw
```

Windows ではダブルクリック起動でも動作します。

## 使い方

1. `押下間隔（秒）` に待機時間を入力します。
2. `ON` を押すと開始します。
3. `停止` を押すと停止します。
4. `終了` を押すとアプリを閉じます。

## ショートカット

- `l + 1`: 開始
- `l + 2`: 停止

## 動作内容

- `Ctrl` を押しっぱなしにします
- `W / A / S / D` のいずれかをランダムで短時間押下します
- 設定した間隔だけ待機して、次の入力を行います

## 注意

- `keyboard` は環境によって管理者権限が必要になる場合があります。
- 実行中はアクティブなウィンドウに対してキー入力が送られます。
- 予期しない操作を避けるため、使用中のアプリに注意してください。

## 補足

このスクリプトは起動時に不足しているパッケージを自動インストールする処理を含んでいます。  
ただし、配布や再現性の観点では `requirements.txt` からのインストールを推奨します。
